# pdfshow

Present PDF slideshows using a browser. Supports writing on slides using a pointer
device (e.g. Surface Pro Pen). Supports Surface Pro Pen eraser.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
